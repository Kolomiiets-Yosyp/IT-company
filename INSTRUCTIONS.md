# Інструкції по запуску та керуванню сайтом Tworix

## 1. Встановлення залежностей

Спочатку вам потрібно встановити всі необхідні залежності. Відкрийте термінал у кореневій папці проекту та виконайте наступну команду:

```bash
pip install -r requirements.txt
```

## 2. Застосування міграцій бази даних

Далі, вам потрібно створити базу даних та застосувати до неї всі міграції:

```bash
python tworix/manage.py migrate
```

## 3. Заповнення бази даних початковим контентом

Щоб сайт виглядав так, як задумано, потрібно заповнити базу даних початковим контентом (тексти, проекти). Для цього виконайте наступні команди:

```bash
python tworix/seed_projects.py
python tworix/seed_content.py
```

## 4. Створення суперкористувача

Для доступу до адмін-панелі вам потрібен обліковий запис суперкористувача. Ви можете створити його за допомогою наступної команди та слідувати інструкціям на екрані:

```bash
python tworix/manage.py createsuperuser
```
**Примітка:** Я вже створив для вас суперкористувача за замовчуванням:
- **Логін:** `admin`
- **Пароль:** `admin`

## 5. Запуск сервера для розробки

Тепер ви готові запустити сайт. Виконайте наступну команду:

```bash
python tworix/manage.py runserver
```

Після цього сайт буде доступний у вашому браузері за адресою: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## 6. Доступ до адмін-панелі та керування контентом

Адмін-панель - це місце, де ви можете редагувати весь контент на сайті.

- **URL для доступу:** [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
- **Логін:** `admin`
- **Пароль:** `admin` (або той, який ви створили на кроці 4)

Після входу ви побачите три основні розділи:

- **Contents:** Тут ви можете редагувати весь текстовий контент на сайті. Кожен запис має `ключ` (який використовується в коді, його не варто змінювати) та `значення` (сам текст, який ви можете редагувати).
- **Projects:** У цьому розділі ви можете додавати, редагувати та видаляти проекти, які відображаються на сторінці "The Lab". Ви можете завантажувати зображення, вказувати назви, описи та технології.
- **Contacts:** Тут зберігаються всі повідомлення, надіслані через контактну форму на сайті.

Просто знайдіть потрібний розділ, виберіть елемент для редагування, внесіть зміни та натисніть "Save". Зміни одразу ж з'являться на сайті.
